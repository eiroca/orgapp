README
======
To build the orgapp application from source, you will need:

- Ant (http://ant.apache.org)
- Proguard (http://proguard.sourceforge.net/)

1) Copy the file sample-build.properties to build.properties 
2) Edit the build.properties file and set values according to your environment (proguard.home should point to Proguard path, etc).
3) Run Ant (i.e.: at the command line, type: ant)
